## About This Project

This repository contains the source code for a freelance M365 specialist's professional digital CV and portfolio. It demonstrates technical expertise in Microsoft 365 technologies while serving as a functional job application tool.

**Key Features:**
- Responsive design (mobile-first approach)
- Interactive timeline and skill showcase

- GitHub Pages hosting (free, fast, reliable)

---

## Responsive Design

The site is fully responsive across all device sizes:

| Device | Breakpoint | Features |
|--------|-----------|----------|
| **Mobile Portrait** | < 480px | Single-column, stacked buttons, large touch targets |
| **Mobile Landscape** | 480-720px | Two-column layout, flexible spacing |
| **Tablet** | 720-1024px | Three-column sections, responsive cards |
| **Desktop** | 1024px+ | Full layout, hover effects, optimized spacing |

Test responsiveness: Use browser DevTools (F12) and Responsive Design Mode

---

## Accessibility

This site meets **WCAG 2.1 Level AA** standards:

- Semantic HTML5 structure
- Sufficient color contrast (4.5:1 for normal text)
- Keyboard navigation support
- ARIA labels and descriptions
- Focus indicators on interactive elements
- Responsive text sizing
